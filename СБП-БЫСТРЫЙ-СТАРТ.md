## 🔧 Что нужно изменить в коде:

### В `.env` добавить:
```env
# СБП настройки (получите в банке)
SBP_MERCHANT_ID=ваш_merchant_id
SBP_SECRET_KEY=ваш_secret_key
SBP_API_URL=https://api.bank.ru/sbp  # URL от банка
```

### Обновить `paymentService.js`:
```javascript
// Вместо тестового QR-кода делать реальный запрос к банку
async createSbpQrCode(amount, description, orderId) {
  const response = await fetch(`${this.sbp.apiUrl}/qr/create`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${this.sbp.secretKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      merchant_id: this.sbp.merchantId,
      amount: amount,
      order_id: orderId,
      description: description
    })
  });
  
  const data = await response.json();
  return {
    success: true,
    qr_code: data.qr_string,
    qr_id: data.qr_id,
    expires_at: data.expires_at
  };
}
```
