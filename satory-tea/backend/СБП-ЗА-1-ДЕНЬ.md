# 🚀 Запуск СБП за 1 день

## ⚡ Быстрый план:

### **Утром (30 минут):**
1. Перейти на https://www.tinkoff.ru/business/acquiring/sbp/
2. Заполнить заявку (ИП или ООО)
3. Указать сайт: `https://satory-tea.ru`
4. Отправить заявку

### **Днем (получить доступы):**
Тинькофф пришлет SMS/email с:
- `MERCHANT_ID` 
- `SECRET_KEY`
- `API_URL`

### **Вечером (15 минут настройки):**

1. **Обновить .env файл:**
```env
# СБП настройки от Тинькофф
SBP_MERCHANT_ID=ваш_реальный_merchant_id
SBP_SECRET_KEY=ваш_реальный_secret_key  
SBP_API_URL=https://securepay.tinkoff.ru/sbp/v1

# Домен с HTTPS (обязательно!)
APP_URL=https://satory-tea.ru
```

2. **Перезапустить сервер:**
```bash
npm restart
# или
pm2 restart all
```

3. **Протестировать:**
- Создать тестовый заказ
- Выбрать "СБП" 
- Получить QR-код
- Оплатить через банковское приложение

---

## 📱 Как тестировать:

### **В мобильном приложении банка:**
1. Открыть камеру или раздел "Платежи"
2. Сканировать QR-код
3. Подтвердить платеж
4. Деньги придут мгновенно

### **Тестовые суммы:**
- 100₽, 500₽, 1000₽
- Комиссия: 0.4% (4₽ с 1000₽)
