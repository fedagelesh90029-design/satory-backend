# Исправление ошибки авторизации по SMS/Telegram

## Проблема
При попытке получить код подтверждения возникает ошибка, потому что:
- `SMS_PROVIDER=mts` но ключ МТС не настроен
- Telegram бот уже работает, но не используется для SMS

## Решение

### 1. Обновить переменные на Railway

Зайдите в Railway → ваш проект → Variables и измените:

```
SMS_PROVIDER=telegram
```

Убедитесь что есть:
```
TG_BOT_TOKEN=8064093523:AAFF8c8lUSBhoEH-He7ddMAAVfsXLKKu7_U
APP_URL=https://satory-backend-production.up.railway.app
```

### 2. Перезапустить сервис

После изменения переменных Railway автоматически перезапустит сервис.

### 3. Как это работает

Теперь при нажатии "Получить код по SMS":
- Если у пользователя уже привязан Telegram → код придет в бот
- Если не привязан → откроется бот для привязки, затем код придет туда

Кнопка "Получить код в Telegram" работает так же.

## Альтернатива: Настроить МТС Exolve

Если хотите реальные SMS:

1. Зарегистрируйтесь на https://exolve.ru
2. Получите API ключ
3. Добавьте в Railway:
   ```
   SMS_PROVIDER=mts
   MTS_API_KEY=ваш_ключ
   MTS_SENDER=SatoriTea
   ```

Стоимость: ~1.2₽ за SMS
